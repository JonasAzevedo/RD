WebSnap Demos
ScriptObjectTest Directory

This example uses the sample components in the DclSampleScriptObject package.
The SampleScriptObject, ExtendedAapter, and ExtendedDataSetAdapter demonstrate
how to develop Active Script enabled components.

The Home Web Page Module contains instances of these components.  JavaScript in
homeu.html displays the values of their SampleValue properties.

Before using this example:

  run tregsvr demos\websnap\scriptobject\sample.tlb to register the typelib.

  Install demos\websnap\scriptobject\dclsamplescriptobject.bpl.

See demos\websnap\scriptobject\readme.txt for more information.
