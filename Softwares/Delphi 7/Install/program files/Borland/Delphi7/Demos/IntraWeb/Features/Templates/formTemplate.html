Template Example

This page is a demonstration of an HTML template. You can put all of your program's logic, dynamic controls, and input controls in your Delphi application but all your HTML layout, graphics and other in an HTML file.

This allows you to separate the presentation from your application. It also allows your web designers or graphic artists to use any web design tool they want, and let you go back to programming. :)

You can use one template, or you can even have seperate templates for Internet Explorer and Netscape.
